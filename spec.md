# 내 차 유지비 계산기 — Specification

## 1. 개요

| 항목 | 내용 |
|------|------|
| 프로젝트명 | 내 차 유지비 계산기 |
| 목적 | 차종·연비·주행거리·보험료 입력 → 월간/연간 유지비 자동 산출 |
| 타깃 사용자 | 차량 보유자, 차량 구매 검토 중인 사용자 |
| 배포 타깃 | Vercel |

---

## 2. 사용자 스토리

| ID | 스토리 | 우선순위 |
|----|--------|----------|
| U1 | 차종을 선택해서 정기점검비 기준을 자동 반영받고 싶다 | 필수 |
| U2 | 연료 종류(휘발유/경유/전기)를 선택해서 유가 기준을 자동 적용받고 싶다 | 필수 |
| U3 | 월 평균 주행거리와 연비를 입력하면 주유비를 계산받고 싶다 | 필수 |
| U4 | 보험료를 선택적으로 입력해서 합산 금액에 포함시키고 싶다 | 선택 |
| U5 | 결과를 항목별(주유비·보험료·점검비)로 분리해서 보고 싶다 | 필수 |
| U6 | 결과 페이지에서 다시 계산 시 이전 입력값이 유지되길 원한다 | 필수 |
| U7 | 필수 항목 미입력 시 어떤 항목이 문제인지 즉시 알고 싶다 | 필수 |

## 3. 기능 요구사항

### 3-1. 입력 페이지 (`/`)

| 필드 | 타입 | 필수 | 유효성 규칙 |
|------|------|------|-------------|
| 차종 | 선택 (경차/소형/중형/SUV/대형) | ✅ | 1개 선택 |
| 유류 종류 | 선택 (휘발유/경유/전기) | ✅ | 1개 선택 |
| 월 평균 주행거리 | 숫자 입력 (step=100) | ✅ | 빈값 불가 |
| 연비 | 숫자 입력 (step=0.1) | ✅ | 1 이상 |
| 월 보험료 | 숫자 입력 (step=1000) | ❌ | 없으면 0으로 처리 |

**인터랙션:**
- 차종·유류 선택: 토글 버튼 (선택 시 primary 색상 강조)
- 연비 단위: 유류 종류에 따라 `km/L` ↔ `km/kWh` 자동 전환
- 제출 시 유효성 검사 → 오류 시 해당 필드 빨간 테두리 + 메시지

### 3-2. 계산 엔진

```
주유비 = ROUND(주행거리 ÷ 연비 × 유가)
월 합계 = 주유비 + 보험료 + 정기점검비
연 합계 = 월 합계 × 12
```

**기준 유가 (하드코딩)**

| 유료 종류 | 단가 |
|-----------|------|
| 휘발유 | 1,650원/L |
| 경유 | 1,500원/L |
| 전기 | 50원/kWh |

**정기점검비 (월 환산, 하드코딩)**

| 차종 | 금액 |
|------|------|
| 경차 | 20,000원 |
| 소형 | 30,000원 |
| 중형 | 40,000원 |
| SUV | 50,000원 |
| 대형 | 60,000원 |

### 3-3. 결과 페이지 (`/result`)

| 구성 요소 | 설명 |
|-----------|------|
| 월간 합계 배너 | primary 색상 강조, 연간 환산 표시 |
| 항목별 breakdown | 주유비·보험료·정기점검비 개별 표시 |
| 연간 유지비 카드 | `월 합계 × 12` 계산 표시 |
| 다시 계산하기 버튼 | 이전 입력값을 URL 파라미터로 유지하며 `/`로 이동 |
