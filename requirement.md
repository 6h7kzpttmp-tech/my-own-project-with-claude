# 🚗 내 차 유지비 계산기 — Requirements

## 1. 프로젝트 개요

**목표**: 차종·연비·주행거리·보험료를 입력하면 월간/연간 유지비를 추정해주는 웹앱  
**배포 타깃**: Vercel  
**완주 목표 시간**: 3시간 이내  

---

## 2. 페이지 구성 (2페이지)

### Page 1 — 입력 페이지 (`/`)
- 차종 선택 (경차 / 소형 / 중형 / SUV / 대형)
- 월 평균 주행거리 입력 (km)
- 연비 입력 (km/L)
- 유류 종류 선택 (휘발유 / 경유 / 전기)
- 월 보험료 입력 (원, 선택)
- **"계산하기"** 버튼

### Page 2 — 결과 페이지 (`/result`)
- 월 유지비 합계 (강조 표시)
- 항목별 breakdown:
  - 주유비
  - 보험료
  - 정기점검 예상비 (차종별 고정 추정값)
- 연간 유지비 환산
- "다시 계산하기" 버튼

---

## 3. 핵심 기능 (1개)

| 기능 | 설명 |
|------|------|
| **유지비 계산 엔진** | 입력값 → 주유비(주행거리/연비×유가) + 보험료 + 정기점검비 추정 → 월/연 합계 출력 |

> 기준 유가 (하드코딩 초기값): 휘발유 1,650원/L · 경유 1,500원/L · 전기 50원/kWh  
> 정기점검비 추정: 경차 2만 · 소형 3만 · 중형 4만 · SUV 5만 · 대형 6만 (월 환산)

---

## 4. 시나리오 (6개)

| # | 시나리오 | 입력 | 기대 결과 |
|---|----------|------|-----------|
| S1 | 정상 계산 — 중형 휘발유 | 중형, 1,500km, 12km/L, 보험 8만 | 주유비 ~20.6만 + 보험 8만 + 점검 4만 = 약 32.6만 |
| S2 | 경차 전기차 계산 | 경차, 800km, 6km/kWh, 보험 4만 | 전기 요금 ~0.67만 + 보험 4만 + 점검 2만 = 약 6.7만 |
| S3 | 보험료 미입력 | 중형, 1,000km, 11km/L, 보험 없음 | 보험 0원으로 처리, 나머지만 계산 |
| S4 | 필수 입력 누락 | 주행거리 미입력 후 계산 버튼 클릭 | 해당 필드 빨간 테두리 + "필수 입력값입니다" 메시지 |
| S5 | 연비 0 또는 음수 입력 | 연비에 0 입력 | "1 이상의 값을 입력해주세요" 유효성 오류 |
| S6 | 결과 → 재계산 | 결과 페이지에서 "다시 계산하기" 클릭 | 입력 페이지로 이동, 이전 값 유지 |
